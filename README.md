# QuickCrave

### 使用前请先确保node.js版本在18.0 及以上

```
|- android # android 项目目录
|- ios # iOS 项目目录
|- src # 开发主文件夹
|   |- common # 静态资源
|   |- components # 一些公共组件
|   |- services # 前端接口请求
|   |- pages # 页面目录
|   |- App.js # 入口文件
|- index.js # 入口文件
```
---

# 快速启动

```
cd client
npm install -s
npm install -g yarn
```
```
yarn android
```
